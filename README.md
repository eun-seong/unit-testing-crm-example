# 고객 관리 시스템

## 요구 사항

1. 사용자 이메일이 회사 도메인에 속한 경우 `직원`으로 표시, 아닌 경우 `고객`
2. 시스템은 회사의 직원 수를 추적해야 한다. 사용자 유형이 변경될 경우, 직원 수도 변경해야 한다.
3. 이메일이 변경되면 시스템은 메시지 버스로 메시지를 보내 외부 시스템에 알려야 한다.
